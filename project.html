File I/O and File Copy Operations

Programming Language: C
File I/O and file copy operations project is a workshop project from Operating Systems Programming unit. The project retrieves two arguments from the user. First being the file to be copied and second being name for the copied file. File is then copied to same directory with a different i-node number and same file permissions. Error checking is in place.
